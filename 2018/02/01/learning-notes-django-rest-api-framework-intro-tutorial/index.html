前言

隨著 Web 與行動應用的發展，API（應用程式介面）已成為前後端溝通的橋樑。RESTful API 是目前最常見的 API 設計風格之一，而 Django REST Framework（簡稱 DRF）則是基於 Django 的強大工具，讓開發 RESTful API 變得更簡單。

本篇筆記將帶我們從零開始，快速建立一個 Django RESTful API 專案，實作一個基本的「文章系統」。
