1. Node.js 簡介

Node.js 是一個基於 Chrome V8 JavaScript 引擎的運行環境，可讓開發者使用 JavaScript 來撰寫後端程式。它適合用於構建高效能、非同步的網路應用。

1.1 為何選擇 Node.js？

  • 非同步 & 事件驅動：適合 I/O 密集型應用，如 Web 伺服器。
  • 單一語言開發：可用 JavaScript 同時開發前端與後端。
  • 強大的生態系統：擁有豐富的 NPM（Node Package Manager）套件。
  • 高效能：基於 V8 引擎，運行速度快。
