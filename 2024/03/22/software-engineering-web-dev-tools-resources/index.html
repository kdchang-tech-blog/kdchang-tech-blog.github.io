VS Code 排版工具

安裝 Prettier 外掛
1️. 打開 VS Code → Extensions → 搜尋 Prettier - Code formatter → 安裝

2️. 在你的 VS Code 設定檔 .vscode/settings.json（或直接設定 → 搜尋「prettier」）加入：

prettier.semi: true 就是 每行最後自動加分號
以後每次 存檔（Ctrl+S）會自動補分號、排版！
